# 视频服饰植入分析器

对用户上传的视频进行智能深度分析，判断是否适合服饰广告植入，并输出推荐服饰风格。

## 核心判断标准

**必要条件：真人出镜**
- 视频中必须有人物出镜才具备服饰植入价值
- 无真人出镜的视频直接判定为"不适合服饰植入"

## 分析维度

### 1. 视频类型识别
- Vlog/生活记录
- 剧情/短剧
- 教程/教学
- 测评/开箱
- 时尚/穿搭
- 运动/健身
- 美食/探店
- 旅行/风景
- 其他

### 2. 出镜人物分析
- 是否有真人出镜（必选项）
- 出镜人数（单人/多人/群像）
- 性别特征（男/女/混合）
- 年龄段判断（青少年18-、青年18-35、中年35-50、中老年50+）
- 现有穿搭风格
- 人物形象气质

### 3. 场景环境分析
- 场景类型（室内/室外/混合）
- 场景风格（现代/复古/自然/都市/其他）
- 光线条件（自然光/人工光/混合）
- 整体色调

### 4. 视频氛围调性
- 整体风格定位
- 情感基调（轻松/严肃/浪漫/活力/其他）
- 视觉质感
- 节奏感

### 5. 受众群体分析
- 目标受众性别
- 目标受众年龄段
- 消费能力预估
- 兴趣标签

## 多模态AI视频分析方法

本技能使用多模态AI对视频进行深度视觉分析，通过提取视频关键帧并进行智能识别。

### 分析流程

#### 第一步：获取视频
- **本地视频文件**：用户上传的视频文件
- **URL链接**：在线视频地址

#### 第二步：提取关键帧
使用以下方法提取视频关键帧：

**方法A：本地视频（ffmpeg提取帧）**
```bash
# 提取视频关键帧（按时间间隔或场景变化）
ffmpeg -i video.mp4 -vf "fps=1/5,scale=1280:-1" -q:v 2 frames/frame_%03d.jpg
# 参数说明：每5秒提取一帧，缩放到1280宽度
```

**方法B：在线视频（浏览器截图）**
- 使用浏览器工具打开视频URL
- 播放并截取关键画面
- 保存为图片进行分析

#### 第三步：多模态AI分析
对提取的关键帧使用多模态AI进行逐项分析：

1. **真人出镜检测**
   - 分析每帧是否包含人物
   - 统计出镜时间和频率
   - 识别人物数量和特征

2. **人物特征识别**
   - 性别判断
   - 年龄段估算
   - 穿搭风格识别
   - 形象气质分析

3. **场景环境识别**
   - 室内/室外判断
   - 场景风格分类
   - 光线和色调分析

4. **视频内容理解**
   - 视频类型判断
   - 情节和氛围分析
   - 受众群体推断

#### 第四步：综合评估与报告生成
基于多模态分析结果，生成完整的服饰植入分析报告。

### 关键帧提取策略

| 视频时长 | 提取策略 | 帧数建议 |
|---------|---------|---------|
| < 1分钟 | 每2秒1帧 | 最多30帧 |
| 1-5分钟 | 每5秒1帧 | 最多60帧 |
| 5-15分钟 | 每10秒1帧 | 最多90帧 |
| > 15分钟 | 每15秒1帧 + 场景变化帧 | 最多120帧 |

### 多模态分析Prompt模板

分析每帧时使用以下提示词：

```
请分析这张视频截图，提取以下信息：
1. 是否有人物出镜？如有，描述人物特征（性别、年龄段、穿搭风格）
2. 场景类型和风格是什么？
3. 整体视觉氛围和色调如何？
4. 这可能是什么类型的视频内容？
```

## 执行流程

1. **获取视频**：接收用户上传的本地视频文件或URL链接
2. **提取关键帧**：
   - 本地文件：使用ffmpeg提取关键帧图片
   - URL链接：使用浏览器工具截图或下载后提取
3. **多模态AI分析**：对关键帧进行视觉分析
4. **真人检测**：首先判断是否有真人出镜
   - 无真人出镜 → 直接输出"不适合"结论
   - 有真人出镜 → 继续多维度分析
5. **综合评估**：根据各维度分析结果，评估植入适合度
6. **风格匹配**：基于分析结果，匹配推荐服饰风格
7. **生成报告**：按照输出格式模板生成完整分析报告

## 辅助工具

### 关键帧提取脚本

本技能提供了关键帧提取脚本 `scripts/extract_frames.py`，用于从视频中提取关键帧。

**使用方法：**
```bash
python scripts/extract_frames.py <视频路径> [输出目录]
```

**依赖要求：**
- Python 3.6+
- ffmpeg（需要预先安装）

**安装ffmpeg：**
```bash
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt install ffmpeg

# Windows
# 从 https://ffmpeg.org/download.html 下载安装
```

### 脚本功能

- 自动根据视频时长计算最优提取间隔
- 智能限制最大帧数，避免过多帧导致分析时间过长
- 输出JSON格式结果，包含帧路径和分析建议

## 注意事项

- 分析应客观中立，如实反映视频情况
- 若视频无法访问或解析失败，需明确告知用户原因
- 对于边界情况（如真人出镜时间很短），应如实说明并给出部分适合的评价
- 推荐的服饰风格应与视频调性、受众特征相匹配
- 植入建议应具体可执行，避免过于笼统
- 需要确保ffmpeg已安装才能正常提取视频关键帧
