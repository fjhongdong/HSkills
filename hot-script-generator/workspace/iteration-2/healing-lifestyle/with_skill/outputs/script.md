# 独居生活治愈瞬间 - 短视频脚本

【脚本类型】氛围感/OOTD

【受众画像】20-30岁女性，慵懒舒适风，居家/休闲场景

【情绪】温暖治愈，安逸自在

【主题】独居生活中的小确幸与自我陪伴

【场景】周末午后，家中窗边，阳光正好

【故事线】阳光透过窗帘洒落（钩子：视觉美学）→ 沉浸阅读时光 → 微风轻抚 → 治愈一笑

---

## 【脚本】（约20秒，适配9:16竖屏）

**镜头1：远景，阳光透过米白色纱帘，女主坐在窗边看书的背影【前3秒钩子：视觉美学】**
- 服饰：米色慵懒风卫衣宽松地包裹着身体，阳光在上面泛起柔光，整体氛围温暖慵懒
- 台词：无

**镜头2：中景，女主低头翻页，手指轻轻划过书页**
- 服饰：卫衣的袖口微微堆叠，露出纤细的手腕，随性自在
- 台词：（内心独白）"一个人的午后，也可以很温柔"

**镜头3：近景，微风吹起窗帘，女主抬眼望向窗外，眼神宁静**
- 服饰：卫衣的帽檐随意耷拉着，阳光在肩头跳跃
- 台词：无

**镜头4：中景，女主微微一笑，重新低头看书**
- 服饰：米色卫衣在逆光中呈现温暖的轮廓，整体画面治愈
- 台词：（内心独白）"独居，是与自己的浪漫"
