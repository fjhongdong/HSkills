## 阶段1：人物特征识别

### 1.1 识别维度

| 识别项 | 说明 | 可能取值 |
|--------|------|----------|
| **性别** | 人物性别 | 男/女/中性 |
| **年龄感** | 人物年龄段 | 学生/职场新人/成熟/中年 |
| **气质类型** | 整体气质 | 温柔/干练/文艺/休闲/活泼/沉稳 |

### 1.2 气质类型对照表

| 气质类型 | 特征描述 | 视觉元素 |
|----------|----------|----------|
| **温柔** | 柔和、亲切、温暖 | 柔软面料、暖色调、圆润线条 |
| **干练** | 利落、专业、自信 | 挺括面料、简洁线条、中性色 |
| **文艺** | 清新、内敛、有品位 | 自然材质、复古元素、淡雅色 |
| **休闲** | 随性、舒适、自在 | 宽松版型、运动元素、基础色 |
| **活泼** | 年轻、有活力、开朗 | 鲜艳色彩、印花图案、运动风 |
| **沉稳** | 成熟、内敛、有分量 | 深色系、质感面料、简约设计 |

## 阶段2：服饰风格分析

### 2.1 分析维度

| 分析项 | 说明 | 示例 |
|--------|------|------|
| **款式** | 服装类型 | 卫衣、风衣、针织衫、连衣裙 |
| **颜色** | 主色调/配色 | 米色、灰色、莫兰迪色系 |
| **材质** | 面料质感 | 棉麻、针织、牛仔、丝绸 |
| **整体调性** | 风格定位 | 慵懒/干练/文艺/休闲 |

### 2.2 服饰风格分类

| 风格类型 | 特征 | 典型单品 | 适合场景 |
|----------|------|----------|----------|
| **慵懒家居风** | 舒适、放松、私密 | 卫衣、针织、睡衣风 | 居家、周末 |
| **通勤职场风** | 干练、专业、都市 | 西装、衬衫、风衣 | 办公室、商务 |
| **文艺森系风** | 温柔、自然、清新 | 棉麻、连衣裙、针织 | 书店、咖啡馆 |
| **休闲街头风** | 随性、活力、年轻 | T恤、牛仔、运动鞋 | 街头、聚会 |
| **优雅知性风** | 成熟、精致、有品位 | 质感外套、真丝 | 画廊、餐厅 |
| **浪漫约会风** | 柔美、精致、女性化 | 裙装、蕾丝、高跟鞋 | 约会、派对 |

### 2.3 服饰风格-场景对照表

| 服饰风格 | 适配场景 | 不适配场景 |
|----------|----------|------------|
| 慵懒家居风 | 家中、阳台、沙发、窗边 | 办公室、商务场合 |
| 通勤职场风 | 办公室、地铁站、写字楼 | 居家、运动 |
| 文艺森系风 | 书店、咖啡馆、公园、画展 | 健身房、夜店 |
| 休闲街头风 | 街头、商场、朋友聚会 | 正式场合、婚礼 |
| 优雅知性风 | 画廊、餐厅、高端场所 | 运动场合、菜市场 |
| 浪漫约会风 | 餐厅、电影院、花店 | 办公室、运动 |

## 阶段3：受众画像匹配

### 3.1 分析维度

| 维度 | 分析内容 |
|------|----------|
| **目标人群** | 性别、年龄段、职业身份 |
| **生活场景** | 服饰适合的穿着场景 |
| **风格定位** | 舒适/精致/随性/温柔/干练 |
| **情绪契合** | 服饰传递的情绪感受 |

### 3.2 受众画像模板

```markdown
| 维度 | 内容 |
|------|------|
| 性别 | [男/女] |
| 年龄段 | [学生/20-30岁/30-40岁/40+] |
| 职业身份 | [学生/职场新人/白领/自由职业/...] |
| 生活场景 | [居家/通勤/约会/休闲/旅行] |
| 风格偏好 | [舒适/精致/随性/温柔/干练] |
| 消费心理 | [性价比/品质/品牌/设计感] |
```

### 3.3 核心原则

**脚本人物必须与服饰受众完全一致**

| 服饰风格 | 脚本人物设定 | 场景选择 |
|----------|--------------|----------|
| 慵懒家居风 | 居家、放松、自在 | 家中、阳台、沙发 |
| 通勤职场风 | 干练、专业、都市感 | 办公室、地铁站、写字楼 |
| 文艺森系风 | 温柔、安静、浪漫 | 书店、咖啡馆、公园 |
| 休闲街头风 | 随性、活力、年轻 | 街头、商场、朋友聚会 |
| 优雅知性风 | 成熟、精致、有品位 | 画廊、餐厅、高端场所 |

**错误示例**：
- 服饰：米色慵懒家居服
- 脚本人物：加班到凌晨的打工人 ❌

**正确示例**：
- 服饰：米色慵懒家居服
- 脚本人物：周末宅家看书的治愈女生 ✅

## 阶段4：季节适配检测

### 4.1 服饰季节属性识别

| 服饰特征 | 春季 | 夏季 | 秋季 | 冬季 |
|----------|------|------|------|------|
| **面料厚度** | 轻薄/中等 | 轻薄透气 | 中等偏厚 | 厚重保暖 |
| **袖型** | 长袖/七分 | 短袖/无袖 | 长袖 | 长袖/加厚 |
| **颜色倾向** | 浅色系/花卉 | 清凉色/亮色 | 暖色/大地色 | 深色/厚重色 |
| **款式特点** | 轻便外套/针织 | 连衣裙/短装 | 风衣/毛衣 | 羽绒服/大衣 |

### 4.2 季节-服饰对照表

| 季节 | 典型服饰 | 关键词 | 场景适配 |
|------|----------|--------|----------|
| 🌸 **春季** | 薄外套、针织衫、衬衫、风衣 | 轻盈、清新、温柔 | 踏青、约会、通勤 |
| ☀️ **夏季** | T恤、连衣裙、短裤、防晒衫 | 清凉、活力、轻盈 | 海边、逛街、日常 |
| 🍂 **秋季** | 毛衣、风衣、卫衣、针织 | 温暖、治愈、层次 | 秋游、咖啡馆、居家 |
| ❄️ **冬季** | 羽绒服、大衣、厚毛衣、围巾 | 保暖、厚重、温暖 | 雪景、室内、节日 |

### 4.3 季节适配检测流程

```
步骤1：识别服饰季节属性
├── 分析面料厚度
├── 分析款式特征
├── 分析颜色调性
└── 输出：服饰适宜季节 = [春季/夏季/秋季/冬季/四季通用]

步骤2：获取当前季节/目标季节
├── 当前日期 → 当前季节
├── 热点内容 → 热点关联季节（如"秋天第一杯奶茶"→秋季）
└── 输出：目标季节 = 当前季节 或 热点季节

步骤3：匹配检测
├── 完全匹配 ✅ → 直接使用
├── 部分匹配 ⚠️ → 提示用户，可调整场景
└── 不匹配 ❌ → 强烈建议更换服饰或调整热点
```

### 4.4 适配结果输出

```markdown
### 季节适配检测结果

**服饰分析**：[服饰名称]
├── 面料：[厚度分析] → [适合季节]
├── 款式：[款式分析] → [适合季节]
├── 颜色：[颜色分析] → [最佳季节]
└── **服饰适宜季节**：[最终判断]

**季节匹配**：
├── 当前季节：[当前季节]
├── 目标季节：[热点季节 或 当前季节]
└── 适配结果：✅完全匹配 / ⚠️部分匹配 / ❌不匹配

**建议**：[如有不匹配，给出调整建议]
```

## 与其他技能的协作

### 输入来源

| 上游技能 | 接收内容 |
|----------|----------|
| **hot-spot-analyzer** | 情绪基调、推演场景、推演风格、共鸣人群 |

### 输出供下游使用

| 下游技能 | 传递内容 |
|----------|----------|
| **script-creator** | 角色形象分析结果（JSON格式） |

### 协作流程

```
hot-spot-analyzer（热点深度分析）
        ↓ 输出：情绪基调、推演场景
        ↓
character-fashion-analyzer（角色形象分析）
        ↓ 输出：角色形象分析结果（JSON）
        ↓
script-creator（脚本创作）
        ↓ 输出：完整脚本
```
