## 【四条核心铁律】（全流程不可突破）

### 铁律一：种草核心铁律
- **全程无服装特写镜头**，服装仅作为故事背景元素
- 服装永远服务于人物与剧情，不得作为画面主体
- ❌ 禁止：服装细节特写、标签展示、质地特写、旋转展示
- ✅ 正确：中远景人物情绪，服饰作为画面自然组成部分

### 铁律二：爆款适配铁律
- 严格适配**抖音9:16竖屏格式**
- **前3秒必须设置强钩子**，抓住用户注意力
- 单条视频时长**严格控制在15-30秒**
- 镜头数量控制在4-8个

### 铁律三：审美合规铁律
- 严格遵循**东方正向审美原则**
- 禁止：过度暴露、负面价值观引导、焦虑贩卖、刻板印象强化
- 提倡：自信、独立、温暖、治愈、积极向上的价值观

### 铁律四：角色一致铁律
- **全流程角色形象高度一致**，不得偏离初始设定
- 人物性格、气质、服饰风格、行为方式必须与输入的角色信息保持一致
- ❌ 禁止：角色行为与设定冲突、气质前后矛盾、服饰风格突变
- ✅ 正确：人物状态变化符合角色调性，情绪表达符合角色气质

## 阶段1：输入整合

### 输入方式

支持两种输入方式：

1. **结构化输入**（推荐）：用户提供完整的 JSON 格式分析数据，包含热点情绪/主题/推演信息 + 角色服饰信息
2. **自由描述输入**：用户直接用自然语言描述话题、情绪、角色、服饰等信息，skill 自行提取关键信息

### 1.0 设计理念：全局合成，不选单条热点

当输入包含多条热点的批量分析数据时，不做"选一条热点"的筛选，而是从整体分析中**综合合成**最优的创作素材。

**为什么不用单热点**：
- 单条热点的推演场景/风格/故事线只有一套，创意空间窄
- 丢失了多条热点交汇的深层规律（如焦虑类和治愈类同时出现 = 用户渴望"焦虑→治愈"的情绪转化）
- 热点之间共享的场景、情绪、受众等交叉数据完全浪费

**核心原则**：
- 覆盖面越广的元素 → 传播潜力越大（多条热点共享的场景 > 单条热点的独特场景）
- 数据中的覆盖范围（如 `hotspots` 数组长度）是天然的权重信号
- 跨情绪聚类组合 → 创造意外感和新鲜感

### 1.1 三维综合评分体系

对每个维度的每个候选项计算综合评分，作为后续选择的基础：

```
综合评分 = 覆盖广度分(40%) + 议题热度分(30%) + 服饰适配分(30%)
```

**覆盖广度分**：候选项覆盖的热点/话题数量 ÷ 总数 × 100。覆盖越多话题，传播面越广，代表"时代情绪"而非"单个话题"。若输入只有单条话题，覆盖广度分默认为 70。

**议题热度分**：
- 有 `heat` 字段时：high=100, medium=60, low=30
- 无热度字段时，按话题常见度和时效性推断

**服饰适配分**：
- 场景类：查"服饰风格→适配场景"映射表，高匹配=100，中匹配=60，不适配=0
- 非场景类（情绪、关键词等）：按服饰调性兼容性评分，兼容=80，中性=50，冲突=20

### 1.2 候选集生成

对每个提取维度，**不选唯一最优项**，而是生成带评分的候选集（Top 3-5），供阶段2的十维度系统从中选择和组合。这确保了每次生成可以从不同候选中取值，避免雷同。

**从输入数据中提取候选集**：

| 主题内核 | `theme_summary.core_issues[]` + `core_pain_points[]` | 按热度 × 覆盖数排序，取 Top 3 |
| 共鸣人群 | `theme_summary.audience_profiles[]` | 按占比排序，取 Top 2 |

### 1.3 多样性约束：连续生成的差异化保证

当从同一份数据连续生成多条脚本时，通过以下机制确保每条脚本都不同。

#### 1.3.1 多样性衰减

```
多样性系数 = 1 / (1 + 该候选已被使用的次数 × 0.5)
最终得分 = 综合评分 × 多样性系数
```

已被使用过的候选在后续生成中得分降低，自然"挤"向未使用的候选。

#### 1.3.2 连续生成轮换规则

| 生成序号 | 候选选取策略 | 情绪路径 | 场景选择 |
|----------|-------------|---------|---------|
| 第1条 | 各维度取 #1 候选 | 主聚类情绪 | 评分最高场景 |
| 第2条 | 至少2个维度取 #2 候选 | 主聚类情绪（换表达方式） | 换场景，保留情绪内核 |
| 第3条 | 至少3个维度取 #2-#3 候选 | 尝试次聚类情绪 | 跨场景组合 |
| 第4条+ | 全维度强制轮换 | 跨聚类探索 | 强制场景切换 |

#### 1.3.3 跨聚类组合

最有创意的脚本往往来自不同情绪聚类的交叉。从第3条起优先使用跨聚类组合：

| 组合模式 | 聚类A来源 | 聚类B来源 | 效果 |
|----------|-----------|-----------|------|
| 焦虑→治愈转化 | 焦虑类的痛点 | 治愈类的出口 | 情绪弧线丰富，共鸣面最广 |
| 共鸣+力量 | 反思类的矛盾 | 治愈类的价值主张 | 有深度有温度 |
| 孤独→连接 | 独处类的场景 | 社交类的故事线 | 有反差有温暖 |
| 日常→突破 | 生活类的场景 | 励志类的人物状态 | 平凡中见力量 |

### 1.5 场景序列规划

**设计理念**：一条脚本可以有 1-3 个场景，场景变化服务于叙事和情绪，不是为了换而换。

**上游来源**：
- 必需：候选集中的推演场景（Top 3-5，已带三维评分）
- 必需：角色信息·服饰风格分类
- 参考：情绪基调、叙事结构、推演故事线

**场景数量决策**：

| 时长 | 建议场景数 | 判断依据 |
|------|-----------|---------|
| 15秒 | 1个为主 | 时长短，场景切换会显得仓促 |
| 18-20秒 | 1-2个 | 允许一次自然转换，增加视觉层次 |
| 25-30秒 | 2-3个 | 可以有明确的空间叙事弧线 |

**是否使用多场景的判断**：

| 判断条件 | 场景策略 |
|----------|---------|
| 叙事结构为"蒙太奇式" | 必须多场景（蒙太奇天然需要空间跳跃） |
| 推演故事线含时间跨度（早晨→傍晚） | 建议多场景（用空间变化暗示时间流逝） |
| 情绪基调为"反转型" | 建议多场景（用场景切换强化情绪反转） |
| 脚本类型为"旅行/探店" | 必须多场景（类型特征） |
| 脚本类型为"氛围感/OOTD" + 时长 ≤ 18秒 | 单场景为主（氛围感重在沉浸） |
| 其他 | 根据候选集评分灵活决定 |

**场景变换动机表**：

| 变换动机 | 效果 | 适用场景组合示例 |
|----------|------|-----------------|
| 时间流逝 | 一天/一季的时光感 | 早晨窗边 → 傍晚阳台 |
| 情绪转折 | 空间切换强化情绪变化 | 办公室疲惫 → 咖啡馆释然 |
| 动线推进 | 跟随人物移动 | 卧室 → 客厅 → 门口 |
| 内外对比 | 封闭vs开放的反差 | 室内独处 → 户外开阔 |
| 蒙太奇拼接 | 多空间画面营造氛围感 | 书桌/窗边/咖啡店 快速切换 |

**场景序列规划流程**：

```
步骤1：从候选集中筛选与服饰适配的所有场景（适配分 > 0）
步骤2：根据时长和脚本类型确定场景数量（参考上表）
步骤3：如果多场景 → 选择变换动机（参考动机表）
步骤4：安排场景出场顺序（服务于叙事弧线和情绪走向）
步骤5：设计场景间的转场逻辑（动线/时间/情绪驱动）
步骤6：验证每个场景的服饰适配性
步骤7：输出场景序列规划
```

**服饰风格→适配场景 映射表**：

| 服饰风格分类 | 默认适配场景（高匹配） | 扩展适配场景（中匹配） | 不适配场景 |
|--------------|------------------------|------------------------|------------|
| **慵懒家居风** | 居家、阳台、沙发、窗边 | 咖啡店、书店、周末休闲 | 办公室、商务场合 |
| **通勤职场风** | 办公室、地铁站、写字楼 | 商务餐厅、城市街景 | 居家、运动场 |
| **文艺森系风** | 书店、咖啡馆、公园、画展 | 居家窗边、校园 | 健身房、夜店 |
| **休闲街头风** | 街头、商场、朋友聚会 | 咖啡店、公园、校园 | 正式场合、婚礼 |
| **优雅知性风** | 画廊、餐厅、高端场所 | 咖啡店、书店、剧场 | 运动场、菜市场 |
| **浪漫约会风** | 餐厅、电影院、花店 | 咖啡店、公园、街景 | 办公室、健身房 |

**场景冲突处理规则**：

| 冲突类型 | 说明 | 处理策略 | 示例 |
|----------|------|----------|------|
| **完全冲突** | 最高评分场景与服饰完全不适配 | 降级到次高分场景；如全部冲突则调整场景描述保留情绪内核 | 推演要求"健身房"，服饰为"慵懒家居风"→降级到次高分场景 |
| **部分冲突** | 场景与服饰有部分兼容 | 调整场景描述，保留情绪内核 | 推演"户外"，服饰为"通勤职场风"→调整为"商务出行街拍" |
| **无冲突** | 高分场景与服饰高匹配 | 直接匹配使用 | 推演"窗边"，服饰为"慵懒家居风"→直接匹配 |

**规划示例（单场景）**：
```

决策：氛围感/OOTD + 18秒 → 单场景为主
场景序列：[居家窗边角落]（全片）

推荐场景候选集：
├── 首选：居家窗边角落(90) — 高覆盖+高适配
├── 备选1：咖啡馆(60) — 中等覆盖+中适配
└── 备选2：书店窗边(55) — 扩展适配+安静氛围
```

### 2.0 综合评分引擎（智能匹配预处理）

在开始十维度选择前，先判断创意适配度等级，决定离域倾向。

#### 五因素定性判断标准

| 因素 | 权重优先级 | 低特征 | 中特征 | 高特征 |
|------|------------|--------|--------|--------|
| **情绪复杂度** | 1（最高） | 单一情绪 | 复合情绪 | 矛盾情绪 |
| **热点新鲜度** | 2 | 常见话题 | 中等新颖 | 新颖话题 |
| **服饰风格张力** | 3 | 基础款/百搭 | 有风格但不强烈 | 强风格/个性款 |
| **场景约束强度** | 4 | 场景适配范围窄 | 中等范围 | 场景适配范围宽 |
| **受众年龄层** | 5（最低） | 成熟受众（30+） | 混合受众 | 年轻受众（18-25） |

#### 创意适配度等级判定表

**判定逻辑**：按权重优先级依次判断，高权重因素主导结果。

| 判定条件 | 创意适配度等级 | 离域倾向建议 |
|----------|----------------|--------------|
| 情绪复杂度=高 | **大胆** | 突破层鼓励多离域组合，表达层可离域2项 |
| 情绪复杂度=中 且 热点新鲜度=高 | **大胆** | 突破层鼓励多离域组合，表达层可离域2项 |
| 情绪复杂度=中 且 热点新鲜度=中 | **中等** | 突破层可离域2项，表达层可离域1项 |
| 情绪复杂度=中 且 热点新鲜度=低 | **保守** | 突破层可离域1项，表达层不离域 |
| 情绪复杂度=低 且 (热点新鲜度=高 或 服饰风格张力=高) | **中等** | 突破层可离域2项，表达层可离域1项 |
| 情绪复杂度=低 且 热点新鲜度=中 | **保守** | 突破层可离域1项，表达层不离域 |
| 情绪复杂度=低 且 热点新鲜度=低 | **保守** | 突破层可离域1项，表达层不离域 |

#### 判定示例

**示例1**：
- 情绪：治愈中带自足 → 情绪复杂度=中
- 热点：独居生活治愈瞬间 → 热点新鲜度=低（常见话题）
- 服饰：米色慵懒风卫衣 → 服饰风格张力=中

**判定**：情绪复杂度=中 且 热点新鲜度=低 → **保守**
- 突破层：可离域1项
- 表达层：不离域
- 核心层：不离域

**示例2**：
- 情绪：想逃离又不敢失去 → 情绪复杂度=高（矛盾情绪）
- 热点：职场安全感危机 → 热点新鲜度=中
- 服饰：通勤职场风西装 → 服饰风格张力=中

**判定**：情绪复杂度=高 → **大胆**
- 突破层：鼓励多离域组合
- 表达层：可离域2项
- 核心层：不离域

#### 三层离域倾向等级

| 层级 | 离域倾向 | 判定规则 |
|------|----------|----------|
| 核心层 | 极低 | 无论创意适配度如何，核心层都极少偏离，保一致性 |
| 表达层 | 中等 | 保守→不离域，中等→可离域1项，大胆→可离域2项 |
| 突破层 | 高 | 保守→可离域1项，中等→可离域2项，大胆→鼓励多离域组合 |

#### 连续生成离域触发规则

当用户请求连续生成多条脚本时，应用确定性离域触发规则：

| 连续生成序号 | 离域触发规则 |
|--------------|--------------|
| 第1条 | 按创意适配度等级判定 |
| 第2条 | 表达层强制离域1项（轮换） |
| 第3条 | 突破层强制离域组合 |
| 第4条+ | 全层轮换：核心层可微调，表达层换项，突破层换组合 |

### 镜头1（00:00-00:03）| 中景 | 固定镜头 | 🎣 钩子 | 强度：Lv3 | 场景：居家窗边

**分镜概要**：阳光洒落的窗边角落，女生蜷坐在软垫上阅读。以温暖治愈的画面建立氛围，钩子功能在于视觉冲击力。

**画面描述**：
- 构图：三分法构图，人物位于画面右侧1/3处，左侧留出阳光洒落的空间
- 光线：侧逆光，阳光从画面左侧纱帘透入，形成柔和的散射光，色温暖黄（约4000K）
- 色彩：米白色主调，纱帘透出的暖黄光与米色卫衣呼应，整体色调偏暖偏柔
- 场景道具：窗边软垫、翻开的纸质书、纱帘、窗台上一小盆绿植

**镜头运动**：
- 运动方式：固定
- 起止：画面保持不变
- 速度：静态
- 焦点：焦点在人物面部，前景纱帘轻微虚化形成层次

**服饰融入**：米色慵懒风卫衣自然融入暖调画面，宽大的袖口搭在膝盖上，作为"慵懒居家"氛围的视觉组成部分。无特写，不作为画面主体。

**表演指导**：
- 表情：嘴角自然微弯，带着不经意的浅笑
- 肢体：双腿蜷曲侧坐在软垫上，左手托书，右手轻搭书页
- 眼神：低头看书，随后缓缓抬头看向窗外，眼神温柔

**台词**：（内心独白）"这个午后，只属于我"

**声音设计**：
- 环境音：窗外隐约的鸟鸣声、微风拂动纱帘的轻微声响
- 音效：手指翻动书页的轻柔声
- BGM：钢琴旋律渐入，节奏舒缓，60-70bpm
- 静默设计：开场前0.5秒纯环境音，无BGM，营造"安静进入"的感觉

**转场设计**：叠化过渡到镜头2（阳光从书页过渡到脸上的光影）

**后期提示**：
- 调色：暖黄偏胶片质感，高光微泛橘，暗部保留细节
- 速度：正常速度
- 文字/特效：无

**情绪标注**：平静→沉浸

**高潮功能**：视觉冲击型钩子——温暖治愈的画面在一秒内建立情绪基调，抓住注意力
