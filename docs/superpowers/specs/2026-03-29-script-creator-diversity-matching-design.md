# Script-Creator 多样化智能匹配设计规格

## 概述

本设计为 script-creator 技能的"阶段2：十维度多样化选择"流程增加智能匹配机制，实现：
- 上游分析结果（热点分析 + 角色分析）作为推荐优先级，而非被动参考
- 分层推荐结构，核心维度保一致性，创意维度鼓励突破
- 综合评分计算离域概率，智能判断创意适配度
- 四层面创意提升：维度组合创新、单维度突破、热点解读创意、服饰融入创新

### 三层离域概率范围

| 层级 | 离域概率范围 | 离域含义 |
|------|--------------|----------|
| 核心层 | 5%-15% | 极少偏离，保一致性 |
| 表达层 | 15%-30% | 适度探索，丰富表达 |
| 突破层 | 30%-50% | 鼓励突破，创造惊喜 |

实际离域概率 = 基础值 × 层级系数 + 随机扰动（±5%）

## 综合评分引擎

### 创意适配度评分公式

```
创意适配度分数 = Σ(因素得分 × 权重) / Σ权重
离域概率基础值 = 创意适配度分数 × 3%
```

### 五因素评分标准

| 因素 | 权重 | 评分标准（1-10分） |
|------|------|-------------------|
| **情绪复杂度** | 5 | 1-3分：单一情绪（如"治愈"）<br>4-6分：复合情绪（如"治愈中带自足"）<br>7-10分：矛盾情绪（如"想逃离又不敢失去"） |
| **热点新鲜度** | 4 | 1-3分：常见话题（如"独居治愈"）<br>4-6分：中等新颖（如"职场安全感"）<br>7-10分：新颖话题（如"AI恋爱"、"数字游民"） |
| **服饰风格张力** | 3 | 1-3分：基础款/百搭风格<br>4-6分：有风格但不强烈<br>7-10分：强风格/个性款/独特调性 |
| **场景约束强度** | 2 | 1-3分：场景适配范围窄（如"只能居家"）<br>4-6分：中等范围<br>7-10分：场景适配范围宽（如"居家+咖啡店+公园"） |
| **受众年龄层** | 1 | 1-3分：成熟受众（30+）<br>4-6分：混合受众<br>7-10分：年轻受众（18-25） |

### 评分示例

**输入：**
- 情绪：治愈中带自足（复合情绪）→ 4分
- 热点：独居生活治愈瞬间（常见话题）→ 2分
- 服饰：米色慵懒风卫衣（有风格但不强烈）→ 5分
- 场景适配：居家+周末休闲（中等范围）→ 5分
- 受众：20-30岁女性（年轻受众）→ 8分

**计算：**
```
总分 = (4×5 + 2×4 + 5×3 + 5×2 + 8×1) / 15 = 4.07
离域概率基础值 = 4.07 × 3% = 12.2%

实际离域概率：
- 核心层：12.2% × 0.25 ≈ 3%
- 表达层：12.2% × 1.0 ≈ 12%
- 突破层：12.2% × 2.0 ≈ 24%
```

**解读**：常见热点+基础情绪+中等风格 → 建议保守策略，创意突破适度。

## 设计决策记录

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 智能匹配程度 | 推荐型匹配（B） | 上游分析作为推荐优先级，允许偏离，平衡一致性与创意自由度 |
| 多样化触发时机 | 每次生成时主动选择（A） | 保持当前模式，LLM主动做出多样化选择 |
| 创意度层面 | 四层面（ABCD） | 维度组合创新、单维度突破、热点解读创意、服饰融入创新全覆盖 |
| 匹配逻辑 | 亲和度推荐型（B） | 计算上游特征与维度选项亲和度，按优先级排序推荐 |
| 离域策略 | 智能判断+随机扰动（B+D） | 综合评分计算离域概率基础值，在此基础上随机扰动选择 |
| 创意度判断标准 | 综合评分（D） | 多因素加权评分，适应性强 |
| 评分因素权重 | 情绪复杂度>热点新鲜度>服饰风格张力>场景约束>受众年龄层 | 按用户指定优先级分配权重 |
| 分层结构 | 核心层+表达层+突破层 | 核心保一致性，表达丰富表达，突破鼓励创意 |
| 场景推导 | 仅路径B（智能推导） | 用户可能不提供服饰场景适配，需自动根据热点+服饰风格推导 |
