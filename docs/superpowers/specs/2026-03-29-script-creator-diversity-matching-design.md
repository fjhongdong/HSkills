# Script-Creator 多样化智能匹配设计规格

## 概述

本设计为 script-creator 技能的"阶段2：十维度多样化选择"流程增加智能匹配机制，实现：
- 上游分析结果（热点分析 + 角色分析）作为推荐优先级，而非被动参考
- 分层推荐结构，核心维度保一致性，创意维度鼓励突破
- 综合评分判断创意适配度，定性决定离域倾向
- 四层面创意提升：维度组合创新、单维度突破、热点解读创意、服饰融入创新

---

## 整体架构

### 分层推荐结构

```
上游输入（热点分析 + 角色分析）
        │
        ▼
综合评分引擎
  ├── 判断五因素特征
  ├── 输出创意适配度等级（保守/中等/大胆）
  └── 输出离域倾向判定
        │
        ▼
维度1：脚本类型（独立维度）
  ← 热点推荐脚本类型 + 服饰风格约束
        │
        ▼
第1层：核心维度（一致性层）
  ├── 维度4：情绪基调
  ├── 维度5：视觉风格
  └── 场景设定
  【离域倾向：极低，极少偏离】
        │
        ▼
第2层：表达维度（表达层）
  ├── 维度2：开场钩子
  ├── 维度3：叙事结构
  ├── 维度8：节奏模板
  └── 维度7：台词风格
  【离域倾向：中等，适度探索】
        │
        ▼
第3层：创意维度（突破层）
  ├── 维度10：创新元素
  ├── 维度9：完播钩子
  └── 维度6：镜头语言
  【离域倾向：高，鼓励突破】
        │
        ▼
输出：十维度选择结果 + 多样化维度报告
```

### 三层离域倾向等级

| 层级 | 离域倾向 | 判定规则 |
|------|----------|----------|
| 核心层 | 极低 | 无论创意适配度如何，核心层都极少偏离，保一致性 |
| 表达层 | 中等 | 根据创意适配度等级决定：保守→不离域，中等→可离域1项，大胆→可离域2项 |
| 突破层 | 高 | 根据创意适配度等级决定：保守→可离域1项，中等→可离域2项，大胆→鼓励多离域组合 |

## 综合评分引擎（定性判断版）

### 五因素定性判断标准

| 因素 | 权重优先级 | 低特征（1-3分） | 中特征（4-6分） | 高特征（7-10分） |
|------|------------|-----------------|-----------------|------------------|
| **情绪复杂度** | 1（最高） | 单一情绪（如"治愈"） | 复合情绪（如"治愈中带自足"） | 矛盾情绪（如"想逃离又不敢失去"） |
| **热点新鲜度** | 2 | 常见话题（如"独居治愈"） | 中等新颖（如"职场安全感"） | 新颖话题（如"AI恋爱"、"数字游民"） |
| **服饰风格张力** | 3 | 基础款/百搭风格 | 有风格但不强烈 | 强风格/个性款/独特调性 |
| **场景约束强度** | 4 | 场景适配范围窄（如"只能居家"） | 中等范围（如"居家+咖啡店"） | 场景适配范围宽（如"居家+咖啡店+公园"） |
| **受众年龄层** | 5（最低） | 成熟受众（30+） | 混合受众 | 年轻受众（18-25） |

### 创意适配度等级判定表

**判定逻辑**：按权重优先级依次判断，高权重因素主导结果。

| 判定条件 | 创意适配度等级 | 离域倾向建议 |
|----------|----------------|--------------|
| 情绪复杂度=高 | **大胆** | 突破层鼓励多离域组合，表达层可离域2项 |
| 情绪复杂度=中 且 热点新鲜度=高 | **大胆** | 突破层鼓励多离域组合，表达层可离域2项 |
| 情绪复杂度=中 且 热点新鲜度=中 | **中等** | 突破层可离域2项，表达层可离域1项 |
| 情绪复杂度=中 且 热点新鲜度=低 | **保守** | 突破层可离域1项，表达层不离域 |
| 情绪复杂度=低 且 (热点新鲜度=高 或 服饰风格张力=高) | **中等** | 突破层可离域2项，表达层可离域1项 |
| 情绪复杂度=低 且 热点新鲜度=中 | **保守** | 突破层可离域1项，表达层不离域 |
| 情绪复杂度=低 且 热点新鲜度=低 | **保守** | 突破层可离域1项，表达层不离域 |

### 判定示例

**示例1**：
- 情绪：治愈中带自足 → 情绪复杂度=中
- 热点：独居生活治愈瞬间 → 热点新鲜度=低（常见话题）
- 服饰：米色慵懒风卫衣 → 服饰风格张力=中
- 场景适配：居家+周末休闲 → 场景约束强度=中
- 受众：20-30岁女性 → 受众年龄层=高

**判定**：情绪复杂度=中 且 热点新鲜度=低 → **保守**
- 突破层：可离域1项
- 表达层：不离域
- 核心层：不离域

**示例2**：
- 情绪：想逃离又不敢失去 → 情绪复杂度=高（矛盾情绪）
- 热点：职场安全感危机 → 热点新鲜度=中
- 服饰：通勤职场风西装 → 服饰风格张力=中
- 场景适配：办公室+地铁 → 场景约束强度=低
- 受众：25-35岁职场人 → 受众年龄层=中

**判定**：情绪复杂度=高 → **大胆**
- 突破层：鼓励多离域组合
- 表达层：可离域2项
- 核心层：不离域

### 连续生成离域触发规则

当用户请求连续生成多条脚本时，应用确定性离域触发规则：

| 连续生成序号 | 离域触发规则 |
|--------------|--------------|
| 第1条 | 按创意适配度等级判定 |
| 第2条 | 表达层强制离域1项（轮换） |
| 第3条 | 突破层强制离域组合 |
| 第4条+ | 全层轮换：核心层可微调，表达层换项，突破层换组合 |

### 场景设定 智能推导规则

**上游来源**：
- 必需：热点分析·推演场景
- 必需：角色分析·服饰风格分类

**匹配逻辑**：

```
步骤1：提取热点推演场景关键词
步骤2：提取服饰风格分类
步骤3：查"服饰风格→适配场景"映射表 → 获取适配场景列表
步骤4：计算热点推演场景与适配场景的兼容性
步骤5：输出推荐场景（兼容性最高的场景）
步骤6：若有冲突，执行场景调整
```

**服饰风格→适配场景 映射表**：

| 服饰风格分类 | 默认适配场景（高匹配） | 扩展适配场景（中匹配） | 不适配场景 |

**场景冲突处理规则**：

| 冲突类型 | 处理策略 | 示例 |
|----------|----------|------|
| **完全冲突** | 热点场景与服饰不适配场景完全重叠 | 提示用户，建议调整热点或服饰，或调整场景保留情绪内核 |
| **部分冲突** | 热点场景与服饰有部分兼容 | 调整场景描述，保留情绪内核 |
| **无冲突** | 热点场景与服饰高匹配场景兼容 | 直接匹配使用 |

## 设计决策记录

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 智能匹配程度 | 推荐型匹配（B） | 上游分析作为推荐优先级，允许偏离，平衡一致性与创意自由度 |
| 多样化触发时机 | 每次生成时主动选择（A） | 保持当前模式，LLM主动做出多样化选择 |
| 创意度层面 | 四层面（ABCD） | 维度组合创新、单维度突破、热点解读创意、服饰融入创新全覆盖 |
| 匹配逻辑 | 亲和度推荐型（B） | 计算上游特征与维度选项亲和度，按优先级排序推荐 |
| 离域策略 | 定性判断+确定性规则 | 改为定性判断而非数值计算，便于LLM执行；连续生成使用确定性离域触发规则 |
| 创意度判断标准 | 定性综合判断 | 按权重优先级依次判断五因素特征，输出创意适配度等级 |
| 评分因素权重 | 情绪复杂度>热点新鲜度>服饰风格张力>场景约束>受众年龄层 | 按用户指定优先级分配权重 |
| 分层结构 | 核心层+表达层+突破层 | 核心保一致性，表达丰富表达，突破鼓励创意 |
| 场景推导 | 仅智能推导 | 用户可能不提供服饰场景适配，需自动根据热点+服饰风格推导 |
| 数值计算 | 改为定性判断 | LLM不擅长精确数值运算，定性判断更可执行 |
| 随机扰动 | 改为确定性规则 | LLM无法真正实现随机性，改为连续生成时的确定性触发规则 |
