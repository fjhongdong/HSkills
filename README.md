# HSkills - Claude Code 专业技能集合

Claude Code 自用 Skills 集合，提供一系列专业领域的 AI 辅助技能扩展功能。

---

## 项目概述

HSkills 是一个专业级 Claude Code Skills 仓库，聚焦于**短视频内容创作**、**服饰时尚**、**AI 视频生成**三大核心领域。每个 Skill 都是独立的功能单元，通过自然语言触发，帮助用户完成特定领域的复杂专业任务。

### 核心价值

- **专业化**：每个技能都经过深度设计，融合领域专业知识
- **流程化**：标准化的工作流程，确保输出质量稳定
- **智能匹配**：支持自然语言触发，自动识别用户意图
- **多模态支持**：支持图片、视频、音频等多模态输入分析

## 技能详细说明

### 1. hot-script-generator（热榜服饰种草脚本生成器）

**定位**：根据抖音热榜内容和角色参考图，生成 15-30 秒服饰场景化种草短视频脚本。

**架构模式**：采用**协调器模式**，内部调用三个子技能：

```
hot-script-generator（一键生成入口）
    │
    ├── hot-spot-analyzer（热点深度分析）
    │   输入：热点话题
    │   输出：情绪分析、主题分析、推演结果、创作建议
    │
    ├── character-fashion-analyzer（角色形象分析）
    │   输入：角色参考图/描述 + 热点分析结果
    │   输出：人物特征、服饰风格、受众画像、创作约束
    │
    └── script-creator（脚本创作）
        输入：热点分析报告 + 角色形象分析结果
        输出：完整脚本 + 质量检查 + 配套输出
```

**使用方式**：
- **一键生成**：直接调用 `hot-script-generator`，自动完成全流程
- **分步执行**：依次调用三个子技能，可在步骤间插入人工审核

**核心特点**：
- 无感种草：服饰自然融入故事，叙事为主，软植入而非硬广
- 情绪深度分析：表层情绪 → 深层情绪 → 情绪矛盾 → 情绪出口
- 主题深度挖掘：话题现象 → 背后议题 → 核心痛点 → 共鸣人群 → 价值主张
- **智能匹配机制**：三层推荐结构 + 亲和度映射 + 创意适配度判定
- 爆款节奏设计：多高潮结构（2-3 个高潮点），符合抖音短视频节奏

**四条核心铁律**：
| 铁律 | 核心要求 |
|------|----------|
| 种草核心 | 全程无服装特写，服饰仅作为故事背景 |
| 爆款适配 | 9:16 竖屏，前 3 秒强钩子，15-30 秒，4-8 个镜头 |
| 审美合规 | 遵循东方正向审美原则 |
| 角色一致 | 全流程角色形象高度一致，不得偏离初始设定 |

**热点深度分析报告**（9 部分）：
| 序号 | 模块 | 内容 |
|------|------|------|
| 1 | 热点信息 | 标题、概述、情绪基调 |
| 2 | 情绪深度分析 | 表层→深层→矛盾→出口 |
| 3 | 主题深度挖掘 | 现象→议题→痛点→人群→价值 |
| 4 | 推演结果 | 场景、风格、故事线、人物状态 |
| 5 | 标签提炼 | 情绪/主题/场景/风格/受众标签 |
| 6 | 关键词提炼 | 核心/情绪/场景/行为/视觉关键词 |
| 7 | 议题分析 | 核心议题、背景、热度、诉求、转化方向 |
| 8 | 题材预测 | 适合脚本类型、传播潜力、爆款可能性 |
| 9 | 创作建议 | 推荐类型、钩子类型、创作要点 |

**输出内容**：
- 热点深度分析报告（必须输出）
- 完整脚本（4-8 镜头，适配 9:16 竖屏，多高潮设计）
- 节奏设计（高潮时间表、情绪曲线）
- BGM 建议、标题建议、话题标签
- 脚本自检报告（六维度评分）

### 1.1 hot-spot-analyzer（热点深度分析器）

**定位**：对批量热点话题进行多维度深度分析，输出综合分析报告。

**核心特点**：
- 批量处理：支持批量热点输入，输出汇总分析报告
- 情绪聚类：识别多条热点的情绪类型和趋势
- 议题聚合：识别共同议题，提炼深层诉求

**分析维度**：
| 维度 | 内容 |
|------|------|
| 情绪深度分析 | 表层情绪 → 深层情绪 → 情绪矛盾 → 情绪出口 |
| 主题深度挖掘 | 现象 → 本质 → 共鸣点 |
| 情绪-主题推演 | 场景、风格、故事线、人物状态 |
| 创作建议 | 推荐脚本类型、钩子类型、创作要点 |

**输出格式**：Markdown 报告 + JSON 结构化数据

### 1.2 character-fashion-analyzer（角色形象分析器）

**定位**：根据角色参考图或描述，分析人物特征、服饰风格、受众画像。

**分析维度**：
| 维度 | 内容 |
|------|------|
| 人物特征 | 性别、年龄感、气质类型 |
| 服饰风格 | 款式、颜色、材质、整体调性 |
| 受众画像 | 目标人群、生活场景、风格定位 |
| 季节适配 | 服饰季节属性、匹配检测 |

**服饰风格分类**：
| 风格类型 | 适配场景 |
|----------|----------|
| 慵懒家居风 | 居家、阳台、沙发、窗边 |
| 通勤职场风 | 办公室、地铁站、写字楼 |
| 文艺森系风 | 书店、咖啡馆、公园 |
| 休闲街头风 | 街头、商场、朋友聚会 |
| 优雅知性风 | 画廊、餐厅、高端场所 |
| 浪漫约会风 | 餐厅、电影院、花店 |

**核心原则**：脚本人物必须与服饰受众完全一致

### 1.3 script-creator（短视频脚本创作器）

**定位**：根据热点分析报告和角色形象分析结果，生成 15-30 秒服饰种草短视频脚本。

**核心特点**：
- **全局合成输入**：从批量热点报告中综合合成创作素材，不选单条热点，利用跨热点交汇的深层规律
- **三维评分体系**：覆盖广度(40%) + 议题热度(30%) + 服饰适配(30%)，每个维度生成 Top 3-5 候选集
- **十维度多样化选择**：脚本类型、钩子、叙事结构、情绪基调、视觉风格、镜头语言、台词风格、节奏模板、完播钩子、创新元素
- **多场景序列支持**：支持 1-3 个场景的叙事设计，场景变换服务于时间流逝、情绪转折、动线推进等动机
- **多样性约束**：衰减公式 + 连续生成轮换 + 跨聚类组合，确保连续生成不雷同
- **智能匹配机制**：三层推荐结构 + 亲和度映射 + 创意适配度判定
- **四条核心铁律**：种草核心、爆款适配、审美合规、角色一致

**智能匹配流程**：
```
完整批量报告 → 三维评分 → 候选集生成 → 场景序列规划
    → 创意适配度等级（保守/中等/大胆）
    → 十维度选择（含亲和度映射 + 多样性约束）
    → 脚本生成（多场景 + 详细分镜）
    → 六维度质量自检 → 配套输出
```

**输出格式**：
- 脚本元数据（含脚本概要、场景序列、变换动机）
- 分镜总览表（场景、概要、景别、镜头运动、标记、情绪走向）
- 详细分镜（10 字段：分镜概要、画面描述、镜头运动、服饰融入、表演指导、台词、声音设计、转场设计、后期提示、情绪标注）
- 多样化维度报告（含来源和亲和度）
- 脚本自检报告（六维度评分）
- 配套输出（BGM、标题、标签、发布建议）

### 2. video-shot-breakdown（短视频分镜拆解器）

**定位**：对短视频进行深度分析，逐镜头拆解出详细分镜脚本，支持一比一复刻。

**分析方案**：
| 视频情况 | 推荐方案 | 原因 |
|----------|----------|------|
| 时长 ≤ 2分钟 | 直接视频分析 | 完整捕捉运镜、转场、动作连续性、音频 |
| 时长 > 2分钟 | 关键帧分析 | 避免超出模型处理限制 |

**输出 JSON 结构**：
```json
{
  "video_info": { "duration_seconds", "resolution", "aspect_ratio" },
  "video_analysis": {
    "theme": "视频主题",
    "emotion": { "primary", "secondary", "emotion_arc" },
    "video_type": "视频类型",
    "video_style": "视频风格",
    "on_screen_presence": { "has_real_person", "exposure_level" }
  },
  "shot_breakdown": [{
    "shot_id": 1,
    "shot_type": "景别",
    "camera_movement": "运镜方式",
    "visual": { "scene", "composition", "lighting", "color" },
    "subjects": [{ "description", "action", "clothing" }],
    "audio": { "dialogue", "music", "sound_effects" }
  }]
}
```

**支持识别**：
- 8 种景别（大特写 → 大远景）
- 15 种运镜方式（固定、推、拉、摇、移、跟、升降等）
- 12 种转场类型（硬切、淡入淡出、溶解、匹配剪辑等）

### 3. video-fashion-placement-analyzer（视频服饰植入分析器）

**定位**：分析视频是否适合服饰广告植入，并推荐适合的服饰风格。

**核心判断标准**：
- 必要条件：**真人出镜**（无真人出镜直接判定不适合）

**分析维度**：
1. 视频类型识别（Vlog/剧情/教程/测评/时尚等）
2. 出镜人物分析（人数、性别、年龄段、穿搭风格）
3. 场景环境分析（室内外、风格、光线、色调）
4. 视频氛围调性（情感基调、视觉质感、节奏感）
5. 受众群体分析（目标受众、消费能力、兴趣标签）

**服饰风格分类**：
| 风格类别 | 特点 | 典型单品 |
|----------|------|----------|
| 休闲 | 舒适、随性、日常 | T恤、牛仔裤、卫衣 |
| 商务 | 正式、专业、稳重 | 西装、衬衫、皮鞋 |
| 运动 | 活力、功能性 | 运动服、瑜伽裤、跑鞋 |
| 街头 | 潮流、个性、前卫 | 潮牌T恤、工装裤、板鞋 |
| 优雅 | 精致、高级、气质 | 连衣裙、高跟鞋、手提包 |
| 国潮 | 中国元素、现代设计 | 汉元素服饰、刺绣单品 |

### 4. clothing-analyzer（服饰分析与搭配助手）

**定位**：分析服饰图片，识别受众特征，搜索流行趋势，提供详细搭配建议。

**工作流程**：
1. **服饰分析**：识别款式、颜色、版型、材质、风格特征
2. **受众分析**：性别、年龄段、风格标签、适合场合
3. **趋势调研**：联网搜索当前流行趋势、颜色搭配趋势
4. **搭配建议**：提供多套搭配方案，包含具体单品和搭配逻辑

**输出格式**：
```
## 衣服分析
[款式、颜色、版型、材质、风格特征]

## 受众分析
**性别**：... **年龄段**：...
**风格标签**：#标签1 #标签2 #标签3
**受众画像**：[一句话总结]

## 流行趋势
[当前服饰趋势、颜色搭配趋势、与这件衣服的相关性]

## 搭配建议
### 搭配方案一：[风格名称]
[上装/下装/鞋子/配饰的具体推荐 + 搭配逻辑]
```

### 5. outfit-character-generator（服饰角色生成器）

**定位**：根据服饰图片生成完整角色设定和 AI 绘图提示词。

**工作流程**：
1. **服饰深度分析**：风格定位、受众特征、气质匹配
2. **明星搜索与分析**：搜索当红明星特征，提炼融合
3. **角色设定生成**：完整角色设定卡（外貌、性格、背景）
4. **AI 绘图提示词**：Midjourney / Stable Diffusion 格式提示词

**角色设定卡内容**：
- 基础档案（姓名、年龄、身高、职业）
- 外貌特征（面部、身材、发型、标志性特征）
- 性格特质（核心性格、MBTI、优缺点）
- 气质风格（整体气质、风格关键词、魅力点）
- 穿搭风格（日常风格、本次穿搭解析）
- 生活背景（居住城市、生活方式、兴趣爱好）
- 职业背景、故事背景

### 6. music-matcher（音乐匹配器）

**定位**：分析短视频脚本，从本地音乐库或 API 匹配最适合的背景音乐。

**音乐库输入方式**：
- **本地文件夹**：扫描目录，支持 MP3/WAV/FLAC/AAC/M4A/OGG
- **API 接口**：调用音乐服务 API 获取音乐列表

**匹配评分维度**：
| 维度 | 权重 | 匹配逻辑 |
|------|------|----------|
| 情绪匹配 | 40% | 音乐情绪与脚本情绪基调一致 |
| 节奏匹配 | 30% | BPM 与叙事节奏匹配 |
| 风格匹配 | 20% | 音乐风格与内容类型协调 |
| 时长适配 | 10% | 音乐时长与脚本预估时长接近 |

**本地音频分析能力**：
- BPM 节奏分析（使用 librosa）
- 能量分析（RMS 能量计算）
- 调性分析（大调/小调）
- 情绪预测（综合特征推断）

**支持多模态 AI API**：Claude、OpenAI GPT-4o、自定义 API

### 7. github-repo-search（GitHub 开源项目搜索助手）

**定位**：从用户自然语言需求出发，输出结构化推荐报告。

**工作流程**：
1. **需求收敛**（必须完成）：主题、数量、最低 stars、排序模式、目标形态
2. **检索执行**：5-10 组检索词拆解、候选召回、去重过滤
3. **质量精炼**：噪音剔除、相关性重排、仓库类型分类
4. **交付报告**：Top N 表格，包含项目介绍、推荐理由

**仓库归属类型分类**：
- 通用框架层
- 应用产品层
- 记忆层/上下文基础设施
- MCP 服务层
- 目录清单层（awesome/curated）
- 垂直场景方案层
- 方法论/研究层

**默认参数**：最低 stars ≥ 100、默认 Top 10、archived=false

### 8. script-to-video-prompts（短剧剧本转AI视频提示词生成器）

**定位**：将短剧剧本文档智能拆解为可直接用于 AI 视频生成的完整提示词体系。

**支持的剧本格式**：
| 格式 | 解析方式 |
|------|----------|
| .txt / .md | 直接解析 |
| .docx | python-docx 解析 |
| .pdf | pdfplumber 解析 |
| .fdx | XML 解析 |

**工作流程**：
```
剧本文档 → 剧本解析 → 角色提取 → 场景分析 → 分镜生成 → 一致性校验 → 多格式导出
```

**输出内容**：
1. 项目元数据（片名、集数、总时长、场次数）
2. 风格总设定（画面风格、色彩体系、光影风格）
3. 角色设定库（JSON 结构化 + 自然语言描述）
4. 场景设定库（JSON 结构化 + 自然语言描述）
5. 完整分镜提示词（按场次顺序）
6. 一致性参考表（角色/场景一致性种子词）

**导出格式**：Markdown / JSON / CSV / Excel / HTML

## 使用指南

### 环境要求

- **Claude Code CLI** 或 **VS Code 扩展**
- Python 3.6+（部分技能需要）
- ffmpeg（视频分析相关技能）

### 依赖安装

```bash
# 视频分析依赖
brew install ffmpeg

# 音频分析依赖
pip install librosa numpy scipy

# 文档解析依赖
pip install python-docx pdfplumber

# HTTP 请求
pip install requests
```

### 基本使用

1. 克隆仓库到本地
2. 在 Claude Code 中打开项目目录
3. 通过自然语言触发相应技能

**示例触发方式**：
```
"帮我生成一个抖音种草脚本，热点是【独居治愈】，服饰是米色慵懒风卫衣"
"分析这个视频适合植入什么服饰" [上传视频]
"帮我把这个剧本文档转成AI视频提示词" [上传文档]
"帮我找一个Python的agent记忆库"
```

## 技术亮点

### 1. 多模态 AI 深度应用

- 视频直接分析：完整捕捉运镜、转场、动作连续性
- 音频深度分析：BPM、能量、调性、情绪预测
- 图像语义理解：服饰风格识别、人物特征提取

### 2. 专业领域知识融合

- 影视专业术语：8 种景别、15 种运镜、12 种转场
- 时尚领域知识：10 种服饰风格、年龄段适配、场景匹配
- 内容创作方法论：热点质量评估、情绪深度分析、脚本多样化策略

### 3. 标准化工作流程

每个技能都遵循：
- 明确的输入输出格式
- 结构化的分析维度
- 可复现的工作流程
- 质量检查机制

### 4. 可扩展架构

- 独立的 Skill 定义文件（SKILL.md）
- 可选的自动化脚本（scripts/）
- 规范参考文档（references/）
- 评估测试框架（evals/）
